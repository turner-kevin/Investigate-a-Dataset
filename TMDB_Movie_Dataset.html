Introduction¶

We will be analyzing a dataset from TMDB (The Movie Database). The dataset includes information about 10,000+ movies. The data included in teh dataset contains movie titles, budget, revenue, runtime, genres information, cast, director and much more. The dataset also includes budget and revenue dollars adjusted to 2010 dollars to account for inflation.

Through our analysis of the dataset we will aim to answer two questions:

1. What was the averge earnings of movies made year over year?
2. What was the most produced movie genre for the year I was born?

We start by importing the python libraries that we will use through the analysis of the TMDB Movie Data.

Limitations¶

  • For the purpose of this analysis we only looked at dollar represented values (Budget, Revenue, and the derived Earnings) as well as the information held in the Genre column
  • The dataset contained many zero values for information we would expect to see actual values
    • When all zero values were removed we were left with only approximately 35% of the dataset, possibly removing valid information
  • Dollar values analysed were adjusted for 2010 values to account for inflation in order to make more accurate comparisons
    • This has the effect of dollar values possibly not being accurately represented
  • The Genres column may have contained mutlipe values for each movie
    • All genre values were captured in the analysis therefore movies were counted multiple times when determing percentages of genres produced
